# Tauri + React + Typescript

This template should help get you started developing with Tauri, React and Typescript in Vite.

## macOS 安裝注意事項

由於此 app 未經 Apple 公證，macOS 下載後可能顯示「已損毀，無法開啟」。請在 Terminal 執行以下指令移除隔離屬性後即可正常開啟：

```bash
xattr -dr com.apple.quarantine /Applications/dtd-desktop.app
```
